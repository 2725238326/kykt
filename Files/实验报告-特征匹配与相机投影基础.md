# 实验报告：特征匹配与相机投影基础

## 一、实验名称

基于 SIFT、RANSAC 与相机投影模型的计算机视觉基础实验

## 二、实验目的

本实验的目的在于通过两个基础代码实验，理解计算机视觉中从二维图像到三维几何的关键环节，具体包括：

1. 理解图像特征点提取与匹配的基本流程。
2. 理解 Lowe Ratio Test 在初步筛选匹配中的作用。
3. 理解 RANSAC 与基础矩阵在误匹配剔除中的作用。
4. 理解三维点在相机模型下投影到二维图像平面的过程。
5. 为后续三维重建相关内容打下基础。

## 三、实验内容

本实验主要由两部分组成：

### 1. 双图像特征匹配实验

使用两张具有重叠视角的场景图像，依次完成：

1. 图像读取与灰度化；
2. SIFT 特征点提取；
3. KNN 特征匹配；
4. Lowe Ratio Test 初步筛选；
5. 基于 RANSAC 的基础矩阵估计与误匹配剔除；
6. 匹配结果可视化。

### 2. 虚拟相机投影实验

使用一个简单的立方体三维模型，依次完成：

1. 构造三维点坐标；
2. 构造相机内参矩阵；
3. 设置相机旋转矩阵与平移向量；
4. 将三维点投影到二维图像平面；
5. 可视化二维投影结果与三维场景结构。

## 六、实验原理

### 1. SIFT 特征提取

SIFT 会在图像中寻找具有稳定性的局部关键点，并为每个关键点生成一个描述其邻域纹理信息的描述子。这样，即使两张图像存在一定视角变化，仍然有可能找到对应关系。

### 2. KNN 匹配与 Ratio Test

对一张图中的每个描述子，在另一张图中寻找最接近的两个候选描述子。若最近邻与次近邻的距离差异足够明显，则保留该匹配。这样可以减少区分度不强的错误匹配。

### 3. RANSAC 与基础矩阵

在初步匹配中，仍然会存在部分误匹配。RANSAC 通过随机采样和迭代估计基础矩阵 `F`，利用双视图几何约束判断哪些点满足极几何关系，从而剔除误匹配。

基础矩阵满足的核心约束为：

`x2^T F x1 = 0`

这表示两张图像中的对应点应满足双视图极几何关系。

### 4. 相机投影模型

对于三维点 `P_world`，其在相机坐标系中的表示为：

`P_cam = R * P_world + t`

随后通过内参矩阵 `K` 投影到图像平面，并经过透视除法得到像素坐标：

`u = x' / z'`

`v = y' / z'`

这一过程说明了三维空间中的点如何形成二维图像中的投影。

## 七、实验过程

### 1. 特征匹配实验过程

1. 读取 `2-a.jpg` 和 `2-b.jpg`；
2. 将图像转换为灰度图；
3. 使用 SIFT 提取关键点和描述子；
4. 使用暴力匹配器进行 KNN 匹配；
5. 使用 Lowe Ratio Test 进行初步筛选；
6. 使用 `findFundamentalMat(..., FM_RANSAC)` 估计基础矩阵并剔除误匹配；
7. 绘制并保存 RANSAC 前后的匹配结果图。

### 2. 相机投影实验过程

1. 手工构造一个立方体的 8 个顶点；
2. 设置图像大小和相机焦距；
3. 构造相机内参矩阵 `K`；
4. 设置外参 `R` 和 `t`；
5. 完成三维点到二维像素平面的投影；
6. 可视化二维成像结果和三维场景示意图。

## 八、实验结果

### 1. 特征匹配实验结果

本次使用 `2-a.jpg` 与 `2-b.jpg` 的运行结果如下：

```text
Image 1 keypoints: 105675
Image 2 keypoints: 102141
Matches after ratio test: 1050
Inliers after RANSAC: 732
Rejected ratio: 30.29%
```

实验结果表明：

1. 两张图片中均检测到了大量局部特征点；
2. 经过 Ratio Test 后，保留了 1050 对候选匹配；
3. 经过 RANSAC 几何验证后，最终保留 732 对可信匹配；
4. 约 30% 的候选匹配被判定为误匹配并被剔除。

### 2. 特征匹配结果分析

从匹配图中可以看到：

1. RANSAC 前的匹配中仍存在部分不合理的连线；
2. RANSAC 后保留下来的匹配更加集中在共同可见且纹理稳定的区域；
3. 说明几何约束对于提高匹配可靠性具有明显作用。

### 3. 相机投影实验结果

运行虚拟相机投影代码后，得到的相机内参矩阵为：

```text
[[875.   0. 400.]
 [  0. 875. 300.]
 [  0.   0.   1.]]
```

实验结果说明：

1. 三维立方体可以被成功投影到二维图像平面；
2. 图像中的目标大小与深度存在明显关系；
3. 相机参数决定了最终的成像位置和尺度。

## 九、实验结论

通过本次实验，可以得到以下结论：

1. 基于 SIFT 的局部特征提取能够为双图像匹配提供稳定特征；
2. KNN 与 Lowe Ratio Test 能够完成初步匹配筛选，但仍会保留部分误匹配；
3. RANSAC 与基础矩阵约束可以有效剔除误匹配，提高结果可靠性；
4. 相机投影模型揭示了三维几何结构与二维图像之间的联系；
5. 特征匹配与相机成像模型共同构成了后续三维重建学习的重要基础。
