# 课堂讲解稿（2到3分钟）

大家好，我本次科研课堂项目的主题是计算机视觉中的特征匹配与相机投影基础。我的目标是通过两个小实验，理解从二维图像到三维几何之间的基本联系。

第一部分是双图像特征匹配实验。我选取了两张具有重叠视角的场景图片，也就是 `2-a.jpg` 和 `2-b.jpg`。在这部分实验中，我首先对图像进行灰度化，然后使用 SIFT 算法提取关键点和描述子。关键点可以理解为图像中比较有辨识度的位置，而描述子则是这些位置附近局部纹理的数字表示。

接下来，我使用 KNN 方法进行初始匹配，并通过 Lowe Ratio Test 对匹配结果进行第一轮筛选。这样做的原因是，单纯依赖相似度会产生一些区分度不强的错误匹配。然后，我进一步使用 RANSAC 估计基础矩阵，并利用双视图几何约束剔除误匹配。

在实验结果中，第一张图片检测到 `105675` 个特征点，第二张图片检测到 `102141` 个特征点。经过 Ratio Test 后，保留了 `1050` 对候选匹配；经过 RANSAC 之后，最终保留了 `732` 对内点匹配，剔除率约为 `30.29%`。从 RANSAC 前后的结果图对比中可以明显看到，加入几何约束之后，错误连线减少了很多，匹配结果更加可靠。

第二部分是虚拟相机投影实验。在这个实验中，我构造了一个简单的三维立方体，并设置了相机的内参矩阵、旋转矩阵和平移向量。通过这个过程，我验证了三维点如何先从世界坐标系变换到相机坐标系，再通过相机内参投影到二维图像平面。这个实验帮助我理解了焦距、主点、旋转和平移这些相机参数的物理意义，也理解了为什么图像中会出现近大远小的透视现象。

通过这两个实验，我认识到，特征匹配负责建立图像之间的对应关系，而相机投影模型则解释了二维图像与三维空间之间的几何联系。这两部分内容共同构成了后续三维重建学习的基础。我的后续计划是继续做参数对比实验，并进一步学习基础矩阵、本质矩阵和三角化等内容。

我的汇报结束，谢谢大家。
